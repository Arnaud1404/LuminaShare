Comment setup l'extention Latex : https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop

une fois l'extention installer, aller dans la liste d'extentions puis cliquer sur l'engrenage de l'image de l'extention Latex puis dans settings puis dans la barre de recherche coller ```latex-workshop.latex.autoBuild``` 
Ensuite passer les paramètres : \
```Latex-workshop › Latex › Auto Build: Run``` sur ```on save``` ```Latex-workshop › Latex › Auto Build: Interval``` à ```600```


Pour utiliser l'extention rendez-vous d'ans l'icone qui est apparut en étant sur le .tex puis cliquez sur ``` View Latex PDF ``` puis sur ``` View in Vscode tab```

pour supprimer les dépendences généré aller dans ``` Build LateX project``` puis ``` clean up auxiliary files```
